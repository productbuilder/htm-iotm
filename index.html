IOTM Participation Widget

Embeddable participation flow for collecting community heritage contributions.

Introduction

The widget provides a consistent way to gather local heritage contributions across websites and applications. It helps organizations collect structured submissions without building a custom upload flow for each channel.

What the Widget Does

Contributors can submit heritage material such as:

  • photos
  • videos
  • audio recordings
  • written descriptions

The widget also collects:

  • metadata
  • rights consent
  • contact details

Key Features

  • embeddable web component
  • framework-independent
  • REST API integration
  • configurable text and styling
  • internationalization support
  • accessible design
  • modular upload flow

Integration

The widget can be embedded in:

  • plain HTML pages
  • CMS platforms
  • React applications
  • Vue applications
  • Angular applications
  • server-rendered websites
<iotm-participation-widget
  api-base-url="https://example-api.org"
  project-id="heritage-project"
  locale="en">
</iotm-participation-widget>
