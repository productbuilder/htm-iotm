Timemap Contributor

Embeddable contribution flow for collecting community heritage media and context.

Introduction

Timemap Contributor helps heritage organizations collect structured community submissions without building a custom upload flow for each channel or website.

Plug-and-Play Usage

Load the widget script once, place the custom element on the page, and start collecting contributions. 

<timemap-contributor
  api-base-url="https://example-api.org"
  project-id="heritage-project"
  locale="en">
</timemap-contributor>

What Timemap Contributor Does

Contributors can submit:

  • photos
  • videos
  • audio recordings
  • written descriptions

The flow also gathers:

  • metadata
  • rights consent
  • contact details

Key Features

  • embeddable web component
  • framework-independent
  • REST API integration
  • configurable text and styling
  • internationalization support
  • accessible design
  • modular upload flow
