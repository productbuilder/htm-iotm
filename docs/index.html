Timemap Contributor Documentation

1. Introduction

Timemap Contributor is a Web Component for collecting heritage-related submissions. It can be embedded in any frontend stack that can render HTML custom elements.

2. Installation / Usage

Load the widget module:

<script type="module" src="/src/app/index.js"></script>

Then place the element:

<timemap-contributor></timemap-contributor>

3. Basic Example

<timemap-contributor
  api-base-url="https://example-api.org"
  project-id="heritage-project"
  locale="en">
</timemap-contributor>

4. Attributes

Attribute Description Example
api-base-url Base URL for backend API endpoints. https://example-api.org
project-id Project identifier passed with uploads/submissions. heritage-project
locale Locale used for language resolution. en, nl-NL
theme Theme marker attribute for host styling logic. light
allow-video Enable or disable video uploads. true
allow-audio Enable or disable audio uploads. true
allow-image Enable or disable image uploads. true
max-file-size Maximum file size in bytes per upload item. 10485760
organization-name Organization label shown in widget header/about context. City Archive
header-title Header title shown in the widget shell. Share Your Story

5. Properties

The widget exposes JavaScript properties. A common property for consumers is translations for partial text overrides. 

const widget = document.querySelector("timemap-contributor");

widget.translations = {
  buttons: {
    continue: {
      en: "Continue",
      nl: "Verder"
    }
  }
};

6. Internationalization

  • locale controls active language.
  • English and Dutch are supported.
  • en* resolves to English, nl* to Dutch.
  • Unknown locales fall back to English.
  • Partial translation overrides are merged with defaults.
locale="en"
locale="nl"
locale="en-GB"
locale="nl-NL"

7. Styling

CSS Custom Properties

timemap-contributor {
  --timemap-color-primary: #2b6cb0;
}

CSS Parts

timemap-contributor::part(header) {
  background: #2b6cb0;
}

Exposed parts:

  • shell
  • header
  • sidebar
  • viewport
  • content
  • modal
  • modal-backdrop
  • button
  • input
  • upload-dropzone
